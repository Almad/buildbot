naturalSort(l)

source code 

Returns a sorted copy of l, so that numbers in strings are sorted in the proper order.

e.g. ['foo10', 'foo1', 'foo2'] will be sorted as ['foo1', 'foo2', 'foo10'] instead of the default ['foo1', 'foo10', 'foo2']
